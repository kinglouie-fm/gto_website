## Installation

1. **Clone the repository:**

   ```bash
   git clone https://github.com/your-username/gto-lux-frontend.git
   cd gto-lux-frontend
   ```
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Run the development server:**
   ```bash
   npm run dev
   ```
4. **Build for production:**
   ```bash
   npm run build
   ```
